add_product

UMLOperation
CRM :: Model :: ProductsManager :: add_product

Description

Добавляет новый товар в базу данных.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
link (str): Ссылка на товар.
category_id (int): Идентификатор категории товара.
brand (str): Бренд товара.
model (str): Модель товара.
sku (str): Стоковый номер товара (SKU).
is_visible (bool): Флаг видимости товара.
sections (list): Список идентификаторов разделов, к которым относится товар.
sizes (dict): Список размеров товара. {size: str, price: float)

Returns:

'message': Str,
'product_id': Int
