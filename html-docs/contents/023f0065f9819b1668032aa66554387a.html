mode

UMLParameter
CRM :: Model :: RequestsManager :: set_stat_5 :: mode

Description

Обновляет статус заявки в зависимости от указанного режима. Если mode=True, статус изменяется на "Отправлен СДЭК", а если mode=False – на "Доставлен".

    Args:
        session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
        id_request (int): Идентификатор заявки.
        mode (bool): Режим обновления статуса. True для "Отправлен СДЭК", False для "Доставлен".

    Returns:
        jsonify: Объект JSON с сообщением о результате операции.
