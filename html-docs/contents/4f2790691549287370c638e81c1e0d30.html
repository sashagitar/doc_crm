section_remove

UMLOperation
CRM :: Model :: ProductsManager :: section_remove

Description

Удаляет продукт из раздела, используя идентификаторы продукта и раздела.

Если связь между продуктом и разделом существует, она будет удалена. Если такая связь не найдена, будет возвращено сообщение об ошибке.

Args:

session (Session): Активная сессия базы данных.
prod_id (int): Идентификатор продукта.
section_id (int): Идентификатор раздела, из которого нужно удалить продукт.

Returns:

jsonify: Объект JSON с сообщением о результате операции или ошибке.
