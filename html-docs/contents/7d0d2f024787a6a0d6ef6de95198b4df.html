section_move

UMLOperation
CRM :: Model :: ProductsManager :: section_move

Description

Перемещает товар в другой раздел на основе идентификатора раздела.

Этот метод удаляет существующие связи продукта с разделами и создает новую связь между продуктом и указанным разделом.

Args:

session (Session): Активная сессия базы данных.
prod_id (int): Идентификатор продукта.
section_id (int): Идентификатор целевого раздела для перемещения продукта.

Returns:

'message': Str

Если False:

'error': Str
