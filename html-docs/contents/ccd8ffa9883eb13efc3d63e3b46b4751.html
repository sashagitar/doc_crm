public init(session: Session, ids_requests: list)

Создает новую поставку и обновляет ID поставки для списка заявок.

Эта функция создает экземпляр поставки в базе данных и затем обновляет каждую заявку из предоставленного списка идентификаторов заявок, связывая их с новой поставкой.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
ids_requests (list): Список идентификаторов заявок, которые необходимо связать с новой поставкой.

Returns:

'message': Str
public add(session: Session, shipments_id: int, requests_ids: list)

Привязывает список заявок к определенной поставке, если они еще не привязаны к другой поставке.

Эта функция обновляет каждую заявку из предоставленного списка идентификаторов заявок, присваивая им идентификатор указанной поставки, если они еще не привязаны к другой.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
shipments_id (int): Идентификатор поставки, к которой будут привязаны заявки.
requests_ids (list): Список идентификаторов заявок, которые необходимо привязать к поставке.

'message': Str
