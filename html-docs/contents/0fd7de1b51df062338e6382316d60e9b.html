Description

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.

link (str): Ссылка на товар.
size (str): Размер товара.
cost (float): Стоимость товара.
category_id (int): Идентификатор категории товара.

Returns:

'success':  Bool
'message': Str
'product_id': Int
