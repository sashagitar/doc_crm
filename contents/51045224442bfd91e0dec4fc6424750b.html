get_categories

UMLOperation «»
CRM :: Model :: MarketMngr :: get_categories

Description

Получение списка категорий товаров из базы данных.

Данный метод запрашивает у сессии базы данных все записи категорий и возвращает их в виде списка словарей, содержащих идентификатор и имя категории.

Args:

session (Session): Объект сессии SQLAlchemy для взаимодействия с базой данных.

Returns:

'id': Int
'name': Str
'image_path': Str
