get_brands

UMLOperation
CRM :: Model :: ProductsManager :: get_brands

Description

Получение списка уникальных брендов из базы данных.

Данный метод запрашивает у сессии базы данных уникальные записи брендов из таблицы товаров и возвращает их в виде списка словарей, содержащих название бренда.

Args:

session (Session): Объект сессии SQLAlchemy для взаимодействия с базой данных.
 section_id (int, optional): Идентификатор раздела для фильтрации брендов.
                                 Если None, возвращаются все уникальные бренды.

Returns:

List[Str]
