edit

UMLOperation
CRM :: Model :: BankManager :: edit

Description

Изменяет информацию о банке. Может обновить название или статус видимости.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
bank_id (int): Идентификатор банка.
mode (bool): Режим изменения. Если True, изменяется название, если False - статус видимости.
new_value (str, optional): Новое значение для обновления. По умолчанию None.

Returns: 

'message': Str
