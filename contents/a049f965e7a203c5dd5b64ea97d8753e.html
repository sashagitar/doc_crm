Description

Получение списка закрытых (исторических) заявок пользователя. Функция возвращает информацию о каждой заявке, включая идентификатор заявки, ссылку, бренд, модель, SKU, размер, актуальную цену, и статус доставки.

Args:

session (Session): Сессия для выполнения запросов к базе данных.
tgid (int): Идентификатор пользователя в Telegram.

Returns:

List[Dict[str, Union[int, str]]]: Список закрытых заявок в формате JSON. 
В случае отсутствия пользователя возвращает список с сообщением об ошибке.
