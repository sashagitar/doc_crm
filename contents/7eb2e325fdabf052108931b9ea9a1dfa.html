__check_required_files

UMLOperation «bool»
CRM :: Model :: DeliveriesManager :: «bool»__check_required_files

Description

Проверяет, соответствует ли количество файлов в директории заданному числу.

Args:

request_id (int): Идентификатор заявки.
required_count (int): Требуемое количество файлов в директории.

Returns:

bool: True, если количество файлов соответствует требуемому. В противном случае False.
