Получение списка категорий товаров из базы данных.

Данный метод запрашивает у сессии базы данных все записи категорий и возвращает их в виде списка словарей, содержащих идентификатор и имя категории.

Параметры:

session (Session): Объект сессии SQLAlchemy для взаимодействия с базой данных.

Возвращает:

List[Dict[str, any]]: Список словарей, где каждый словарь представляет категорию.
                    Ключ 'id' хранит идентификатор категории (int),
                    ключ 'name' хранит название категории (str).

public product(session: Session, user_id: int, prod_id: int)

Получение детальной информации о конкретном товаре по его идентификатору для указанного пользователя. Включает в себя проверку на существование пользователя и товара, добавление товара в список просмотренных, и возвращает расширенную информацию о товаре, включая доступные размеры и вкладки с дополнительной информацией.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
tgid (int): Идентификатор пользователя Telegram.
prod_id (int): Идентификатор товара.

Returns:

Dict[str, Union[str, int, List[Dict[str, Union[str, float]]]]]: Словарь с информацией о товаре,
включая его идентификатор, ссылку, бренд, модель, SKU, категорию, доступные размеры и вкладки.
В случае ошибки возвращает сообщение об ошибке.
