filters

UMLParameter
CRM :: Model :: PayManager :: list_payments :: filters

Description

Возвращает список оплат в соответствии с заданными фильтрами.

    Фильтры могут включать номер ссылки, диапазон дат и диапазон цен.

    Args:
        session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
        filters (dict): Словарь с фильтрами для запроса. Ключи могут быть 'ref_number', 'date_from', 'date_to', 'price_from', 'price_to'.

    Returns:
        list: Список словарей с информацией об оплатах, соответствующих фильтрам.
