edit_product

UMLOperation
CRM :: Model :: ProductsManager :: edit_product

Description

Изменяет существующий товар в базе данных.

Args:

session (Session): Сессия

SQLAlchemy для выполнения запросов к базе данных. productid (int): Идентификатор товара. changes (dict): Словарь изменений, содержащий поля для обновления и их новые значения. Может включать: - Основные поля товара (link, categoryid, brand, model, sku, is_visible). - Разделы (sections): Список идентификаторов разделов для обновления.

{ 'brand': 'New Brand', 'model': 'Updated Model', 'sections': [1, 2, 3] # Новые разделы для товара }

Returns:

'message': Str

Если False:

'error': Str
