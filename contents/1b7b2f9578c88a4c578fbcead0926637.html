add_req

UMLOperation
CRM :: Model :: BankManager :: add_req

Description

Добавляет новый реквизит для указанного банка.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
bank_id (int): Идентификатор банка.
card_number (str): Номер карты реквизита.
name (str): Имя владельца реквизита.
wait_time (int): Время ожидания для реквизита.
limit (int): Лимит получения средств на реквизит.

Returns:

Response: Сообщение о добавлении реквизита в формате JSON.
