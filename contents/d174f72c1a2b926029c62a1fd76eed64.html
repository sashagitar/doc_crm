section_move_name

UMLOperation
CRM :: Model :: ProductsManager :: section_move_name

Description

Перемещает товар в другой раздел по названию раздела.

Этот метод удаляет существующие связи продукта с разделами и создает новую связь между продуктом и указанным разделом.

Args:

session (Session): Активная сессия базы данных.
prod_id (int): Идентификатор продукта.
section_name (str): Имя целевого раздела для перемещения продукта.

Returns:

'message': Str

Если False:

'error': Str
