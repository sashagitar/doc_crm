create_payment

UMLOperation
CRM :: Model :: PayMngr :: create_payment

Description

Создание оплаты на основе заявки и реквизита, с возможностью указания шага оплаты.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
request_id (int): Идентификатор заявки.
requisite_id (int): Идентификатор реквизита.
step (float, optional): Шаг оплаты. По умолчанию равен 0.

Returns:

'success': Bool
'message': Str
'payment':  Int

Если False:

'error': Str
