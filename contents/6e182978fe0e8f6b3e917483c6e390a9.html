createrequest

UMLOperation
CRM :: Model :: RequestsManager :: createrequest

Description

Создает новую заявку на покупку товара.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
link (str): Ссылка на товар.
brand (str): Бренд товара.
model (str): Модель товара.
sku (str): Артикул товара.
actual_price (float): Фактическая цена товара.
split (bool): Флаг, указывающий на необходимость разделения доставки.
delivery_type (bool): Тип доставки (True - курьерская, False - почтовая).
address (str): Адрес доставки.

Returns:

jsonify: JSON-объект с сообщением о результате операции и идентификатором новой заявки.
