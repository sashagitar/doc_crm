card_number

UMLParameter
CRM :: Model :: BankManager :: delete_req :: card_number

Description

Изменяет информацию о реквизите в зависимости от указанного режима.

    Args:
        session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
        req_id (int): Идентификатор реквизита, который нужно обновить.
        new_value (str): Новое значение для обновления.
        mode (str): Поле, которое нужно обновить ('card_number', 'name', 'wait_time', 'received_limit', 'is_visible').

    Returns:
        Response: Сообщение об обновлении реквизита в формате JSON.
