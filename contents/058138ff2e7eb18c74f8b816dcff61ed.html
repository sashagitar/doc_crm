Description

Приветственный бонус за использование реферальной ссылки.

Поиск в базе данных пользователя, связанного с предоставленной реферальной ссылкой. В случае успеха пользователю начисляются бонусные баллы, и операция завершается сообщением об успехе. В противном случае возвращается сообщение об ошибке.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
link (str): Реферальная ссылка.

Returns:

Dict[str, Union[bool, str, int]]: Результат операции в формате JSON. В случае успешного добавления бонусов
возвращает ключ 'success' со значением True и сообщение об успехе. В случае ошибки возвращает ключ 'error'   и соответствующее сообщение об ошибке.
