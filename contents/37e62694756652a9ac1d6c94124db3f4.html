get_sections

UMLOperation
CRM :: Model :: ProductsManager :: get_sections

Description

Получение списка разделов товаров из базы данных.

    Данный метод запрашивает у сессии базы данных все записи разделов и
    возвращает их в виде списка словарей, содержащих идентификатор и имя раздела.

Args:

session (Session): Объект сессии SQLAlchemy для взаимодействия с базой данных.

Returns:

List {
    'id': Int
    'name': Str
}
