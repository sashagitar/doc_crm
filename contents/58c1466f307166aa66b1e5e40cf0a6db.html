list_products

UMLOperation
CRM :: Model :: ProductsManager :: list_products

Description

Возвращает список товаров, отфильтрованных по заданным критериям, с учетом коэффициента совпадения.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
filters (dict): Словарь с фильтрами для запроса товаров.
    Может включать:
    - category_ids: Список идентификаторов категорий.
    - is_archived: Флаг архивированности товара.
    - search: Строка для поиска по модели, бренду и названию категории.
page (int): Номер страницы для пагинации.
per_page (int): Количество элементов на странице.
sort_by (str): Поле для сортировки ('relevance', 'sku', 'avg_price', 'title').
order (str): Порядок сортировки ('asc' или 'desc').

Пример использования фильтров:

{
'search': 'Nike Air Max',  # Строка для поиска по модели, бренду и названию категории.
'category_ids': [1, 2, 3],  # Фильтр по категориям
'is_archived': False  # Только неархивированные товары
}

Returns:

dict: Словарь с результатами запроса, содержащий следующие ключи:
    - 'products' (List[Dict]): Список товаров в формате JSON, где каждый элемент содержит следующие поля:
        - 'id' (Int): Идентификатор товара.
        - 'photo' (Str): URL фото товара.
        - 'title' (Str): Название товара (бренд + модель).
        - 'sku' (Str): Артикул товара.
        - 'min_price' (Float): Минимальная цена товара, округленная до двух знаков после запятой.
        - 'max_price' (Float): Максимальная цена товара, округленная до двух знаков после запятой.
        - 'avg_price' (Float): Средняя цена товара, округленная до двух знаков после запятой.
        - 'visibility' (Bool): Флаг видимости товара.
        - 'relevance_score' (Float): Коэффициент совпадения (только если используется параметр 'search').
    - 'total' (Int): Общее количество найденных товаров.
    - 'page' (Int): Номер текущей страницы.
    - 'per_page' (Int): Количество элементов на странице.

Пример результата:

{
"products": [
    {
        "id": 1,
        "photo": "https://example.com/photo1.jpg",
        "title": "Nike Air Max 270",
        "sku": "SKU12345",
        "min_price": 99.99,
        "max_price": 129.99,
        "avg_price": 114.99,
        "visibility": true,
        "relevance_score": 3
    },
],
"total": 100,
"page": 1,
"per_page": 10
}
