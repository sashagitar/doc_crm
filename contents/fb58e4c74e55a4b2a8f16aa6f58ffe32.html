public ref_links(session: Session, tgid: int)

Возвращает все реферальные ссылки пользователя на основе его Telegram ID.

В случае нахождения пользователя в базе, метод собирает и возвращает все его реферальные ссылки. Если пользователь не найден, возвращается соответствующее сообщение об ошибке.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
tgid (int): Telegram ID пользователя.

 'message': Str
public hello(session: Session, link: str, tgid: int)

Приветственный бонус за использование реферальной ссылки.

Поиск в базе данных пользователя, связанного с предоставленной реферальной ссылкой. В случае успеха пользователю начисляются бонусные баллы, и операция завершается сообщением об успехе. В противном случае возвращается сообщение об ошибке.

tgid (int): Telegram ID пользователя.

Returns:

'success' : Bool
'message': Str

Если False

'error' : Str

public history(session: Session, tgid: int, page: int = 1, per_page: int = 10)

Возвращает историю бонусных операций пользователя с учетом пагинации. История включает в себя информацию о транзакциях по начислению и списанию бонусов, связанных с покупками.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
tgid (int): Telegram ID пользователя.

per_page (int, optional): Количество записей на страницу. По умолчанию равно 10.
