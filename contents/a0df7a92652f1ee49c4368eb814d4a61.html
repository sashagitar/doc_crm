get_price_range

UMLOperation
CRM :: Model :: ProductsManager :: get_price_range

Description

Получает минимальную и максимальную цену по всей базе данных или по конкретному разделу.

Args:

session (Session): Объект сессии SQLAlchemy для взаимодействия с базой данных.
section_id (Optional[int]): Идентификатор раздела. Если не указан, возвращает диапазон цен по всей базе данных.

Returns:

'max_price': Float
'min_price': Float
