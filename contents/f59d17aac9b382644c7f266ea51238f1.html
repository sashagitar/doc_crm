create_request

UMLOperation «Dict[str, Union[bool, str]]»
CRM :: Model :: MarketMngr :: «Dict[str, Union[bool, str]]»create_request

Description

Создание новой заявки на товар для пользователя. Опционально можно указать, будет ли использоваться бонус за покупку и необходимость разделения заказа на части.

Args:

session (Session): Сессия для выполнения запросов к базе данных.
tgid (int): Идентификатор пользователя Telegram.
prod_id (int): Идентификатор продукта.
delivery_type_id (int): Идентификатор типа доставки.
split (bool): Необходимость разделения заказа на части.
bonus (bool, optional): Флаг использования бонусов для оплаты. По умолчанию False.

Returns:

Dict[str, Union[bool, str]]: Словарь с результатом создания заявки, содержащий статус операции
и сообщение об успехе или ошибке. В случае ошибки возвращает соответствующее сообщение.
