viewed_products

UMLOperation «»
CRM :: Model :: MarketMngr :: viewed_products

Description

Получение списка просмотренных продуктов пользователя с пагинацией. Включает информацию о каждом просмотренном продукте, такую как идентификатор, модель, бренд, SKU, категорию, и цену.

Args:

session (Session): Сессия для выполнения запросов к базе данных.
tgid (int): Идентификатор пользователя в Telegram.
page (int): Номер текущей страницы для пагинации.
count (int): Количество элементов на страницу.

Returns:

'total_count': Int
'page': Int
'per_page': Int
'viewed_products': List[{
    'id': Int
    'link': Str
    'brand': Str
    'model': Str
    'sku': Str
    'category': Str
    'price': Float
    'images':  Str
}]

Если False:

'error': Str

Parameters

Direction Name Type Description
in session Session

in page int

Возвращает список просмотренных продуктов пользователя с пагинацией.

    Args:

        user_id (int): Идентификатор пользователя.
        page (int): Номер страницы.
        per_page (int): Количество элементов на странице.

    Returns:
        jsonify: Список просмотренных продуктов в формате JSON.
