reset_password

UMLOperation
CRM :: Model :: ModeratorsManager :: reset_password

Description

Сбрасывает пароль модератора, если введённые текущие логин и пароль верны.

Args:

session (Session): Сессия SQLAlchemy для работы с базой данных.
login (str): Логин модератора.
password (str): Текущий пароль модератора.
new_password (str): Новый пароль для модератора.

Returns:

'message': Str

Если False:

'error': Str
