product

UMLOperation «»
CRM :: Model :: MarketMngr :: product

Description

Получение детальной информации о конкретном товаре по его идентификатору для указанного пользователя. Включает в себя проверку на существование пользователя и товара, добавление товара в список просмотренных, и возвращает расширенную информацию о товаре, включая доступные размеры и вкладки с дополнительной информацией.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
tgid (int): Идентификатор пользователя Telegram.
prod_id (int): Идентификатор товара.

Returns:

'id': Int
'link': Str
'brand': Str
'model': Str
'sku': Str
'category': Str
'sizes': {
    'id': Int
    'size': Str
    'price': Float
}
'tabs':  {
    'title': Str
    'description': Str
}
'images':  Str

Если False:

'error': Str
