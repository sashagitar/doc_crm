UMLOperation
CRM :: Model :: RequestsManager :: list

Description

Описание:

Функция возвращает список заявок с учетом различных фильтров и пагинации. Заявки могут быть отфильтрованы по номеру ссылки, имени покупателя, номеру телефона, дате создания, цене, статусу оплаты и типу оплаты (сплит или нет). Также поддерживается сортировка по дате создания.

Аргументы:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
count (int): Количество заявок на страницу.
page (int): Номер страницы.
filters (dict, опционально): Словарь фильтров для применения к запросу. Поддерживаемые фильтры:

python
Copy
{
    'id': int,                                #  id заказа
    'buyer_name': str,           # Имя покупателя (частичное совпадение)
    'buyer_phone': str,          # Номер телефона покупателя (частичное совпадение)
    'date_from': str,            # Начальная дата создания заявки (формат 'YYYY-MM-DD')
    'date_to': str,              # Конечная дата создания заявки (формат 'YYYY-MM-DD')
    'price_from': float,         # Минимальная цена заявки
    'price_to': float,           # Максимальная цена заявки
    'payment_status': str,       # Статус оплаты ("Ожидается" или "Успешно")
    'is_split': bool             # Оплата сплитом (True/False)
}

sort_by (str, опционально): Поле для сортировки. Поддерживаемые значения:

"date_asc" — сортировка по дате создания (от старых к новым).
"date_desc" — сортировка по дате создания (от новых к старым).
"price_asc" — сортировка по сумме оплаты (от меньшей к большей).
"price_desc" — сортировка по сумме оплаты (от большей к меньшей).

Возвращаемые данные: Функция возвращает словарь с результатами, включая список заявок, общее количество заявок, текущую страницу и общее количество страниц.

python
Copy
{
    'results': [
        {
            'id': int,                                     # Номер ссылки
            'payment_status': str,         # Статус оплаты ("Ожидается" или "Успешно")
            'delivery_type': str,               # Тип доставки
            'creation_date': str,              # Дата и время создания заявки (формат 'дд.мм.гггг чч:мм')
            'buyer_name': str,                # Имя покупателя
            'buyer_phone': str,              # Номер телефона покупателя
            'delivery_status': str,          # Статус доставки
            'payment_amount': str,     # Сумма оплаты (формат 'XX XXX ₽')
            'is_split': bool                         # Оплата сплитом (True/False)
        },
        ...
    ],
    'total': int,                         # Общее количество заявок
    'page': int,                          # Текущая страница
    'pages': int                          # Общее количество страниц
}
