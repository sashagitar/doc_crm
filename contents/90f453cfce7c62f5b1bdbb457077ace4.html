Operations

Visibility Name Description
public get_user(session: Session, tg_id: int)

Получение информации о пользователе по его Telegram ID. Включает в себя основные данные пользователя, такие как полное имя, телефон, адрес, статус и адрес пункта выдачи (если задан).

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
tgid (int): Telegram ID пользователя.

Returns:

Dict[str, Union[int, str, None]]: Словарь с информацией о пользователе. В случае отсутствия пользователя
                                возвращает словарь с ключом 'error'.
public set_user(session: Session, tgid: int, username: str = None, phone: str = None, city: str = None, address: str = None, pvz: int = 0)

Обновление информации о пользователе, включая его полное имя, телефон, адрес и пункт выдачи.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
tgid (int): Telegram ID пользователя.
username (str): Полное имя пользователя.
phone (str): Номер телефона пользователя.
address (str): Адрес пользователя.
pvz (int, optional): Идентификатор пункта выдачи. Если не указан, значение не обновляется.

Returns:

Dict[str, Union[bool, str]]: Словарь с результатом операции. В случае успеха содержит сообщение о успешном обновлении.
                            В случае ошибки возвращает сообщение об ошибке.
public create_user(session: Session, tgid: int, login: str, username: str, phone: str, address: str, city: str)

Создание нового пользователя с заданными параметрами. Включает в себя создание реферальной ссылки для нового пользователя.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
tgid (int): Telegram ID пользователя.
login (str): Логин пользователя.
username (str): Полное имя пользователя.
phone (str): Номер телефона пользователя.
address (str): Адрес пользователя.

Returns:

Dict[str, Union[bool, str, int]]: Словарь с результатом операции создания пользователя. В случае успеха содержит сообщение
                                о успешном создании и статус. В случае ошибки возвращает сообщение об ошибке.
public pvz_list(session: Session, city: str = None)

Получение списка пунктов выдачи, опционально фильтруемых по городу.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
city (str, optional): Название города для фильтрации пунктов выдачи. Если None, возвращает все пункты выдачи.

Returns:

List[Dict[str, Union[int, str]]]: Список пунктов выдачи в формате JSON, каждый из которых содержит идентификатор и адрес.
