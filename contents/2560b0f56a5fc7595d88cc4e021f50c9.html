ref_links

UMLOperation «Dict[str, Union[bool, List[str], str]]»
CRM :: Model :: BonusMngr :: «Dict[str, Union[bool, List[str], str]]»ref_links

Description

Возвращает все реферальные ссылки пользователя на основе его Telegram ID.

В случае нахождения пользователя в базе, метод собирает и возвращает все его реферальные ссылки. Если пользователь не найден, возвращается соответствующее сообщение об ошибке.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
tgid (int): Telegram ID пользователя.

Returns:

Dict[str, Union[bool, List[str], str]]: Словарь, содержащий результат операции. В случае успеха
возвращает ключ 'success' со значением True и список ссылок под ключом 'referal_links'.
В случае неудачи возвращает ключ 'success' со значением False и сообщение об ошибке под ключом 'message'.
