UMLOperation
CRM :: Model :: ProductsManager :: info

Description

Получает полную информацию о продукте, включая табы и размеры, по его идентификатору.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
prod_id (int): Идентификатор продукта.

Returns:

Dict:{
    'id': Int,
    'link': Str,
    'brand': Str,
    'model': Str,
    'sku': Str,
    'is_archived': Bool,
    'is_visible': Bool,
    'category_id': Int
    'tabs': [{
        'title': Str,
        'description': Str
        },]
    'sizes': [{
        'size': Str,
        'price': Float
        },]
    }

Если False:

'error': Str
