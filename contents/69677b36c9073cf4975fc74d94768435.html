statistik

UMLOperation «Dict[str, str]»
CRM :: Model :: BonusMngr :: «Dict[str, str]»statistik

Description

Возвращает статистику по бонусной программе для указанного пользователя. Статистика может включать общее количество заработанных и потраченных бонусов, количество выполненных покупок и другие релевантные метрики.

    Args:

        session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
        userid (int): Уникальный идентификатор пользователя в системе.

    Returns:

        Dict[str, str]: Статистика бонусной программы пользователя в формате JSON. В случае отсутствия данных по пользователю может возвращать соответствующее сообщение.
