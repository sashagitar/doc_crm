count_bonus

UMLOperation «»
CRM :: Model :: BonusMngr :: count_bonus

Description

Подсчитывает общее количество бонусных баллов для заданного пользователя.

Вычисляет и возвращает общую сумму бонусных баллов, накопленных пользователем. В случае отсутствия бонусов возвращает ноль.

Args:

session (Session): Сессия SQLAlchemy для выполнения запросов к базе данных.
tgid (int): Telegram ID пользователя.

Returns:

Dict[str, Union[bool, float]]: Словарь с ключом 'success' со значением True и общей суммой бонусов
пользователя под ключом 'total_bonus'.
